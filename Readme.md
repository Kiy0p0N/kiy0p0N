# Felipe Cesar Rodrigues

Desenvolvedor Full Stack com foco no desenvolvimento de aplicações web modernas, escaláveis e centradas na experiência do usuário.

Formado em Análise e Desenvolvimento de Sistemas pela UNINTER e atualmente graduando em Sistemas de Informação na Universidade Federal de Itajubá (UNIFEI).

Tenho experiência no desenvolvimento de aplicações utilizando JavaScript, React, Node.js, Express e PostgreSQL, atuando tanto no frontend quanto no backend. Busco constantemente aprimorar minhas habilidades técnicas e aplicar boas práticas de engenharia de software para construir soluções robustas, eficientes e de fácil manutenção.
